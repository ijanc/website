HTTP / HTTPS

Esse website tem o objetivo de funcionar em computadores antigos bem como acessível por navegadores de terminais. Alguns computadores não suportam protocolo HTTPS por causa dos novos algoritmos ou limitações de memória/CPU. Por padrão o acesso do website é via HTTP mas o mesmo tem suporte completo para HTTPS com classificação A+, para isso basta alterar os links de HTTP para HTTPS.
